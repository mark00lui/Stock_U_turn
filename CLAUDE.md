# Stock U-turn - CTA Trading Signal Dashboard

## Project Overview
CTA (Commodity Trading Advisor) style trading signal dashboard that generates daily Call signals.

## Tech Stack
- Python 3.x
- Data sources: Yahoo Finance (yfinance) or similar
- Dashboard: Streamlit or similar lightweight web framework
- Indicators: Technical analysis libraries (ta, pandas-ta)

## Development Guidelines
- All code in this repo targets Windows 11 environment
- Use UTF-8 encoding throughout
- Git commits should be atomic and descriptive
- Dashboard updates daily - designed for end-of-day signal review

## Git Workflow
- Main branch: `main`
- Auto-commit enabled for development iterations
